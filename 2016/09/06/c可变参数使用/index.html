Posté le 2016-09-06

在C语言编程中有时会遇到一些参数个数可变的函数，例如printf(),scanf()函数，其函数原型为：
int printf(const char format,…)，int scanf(const char format,…);
它除了有一个参数format固定以外，后面跟着的参数的个数和类型是可变的(用三个点“…”做参数占位符)，实际调用时可以有以下的形式: printf(“%d”,i); printf(“%d,%c”,i,j);
